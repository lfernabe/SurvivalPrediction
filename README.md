# SurvivalPrediction
En este repositorio se recoge el código que he utilizado para mi proyecto final de máster de ciencia de datos.

El proyecto está basado en la competición organizada en la plataforma Kaggle por el Center for International Blood and Marrow Transplant Research (CIBMTR).

He dividido el proyecto en 3 scripts:

*  **1_AnálisisVariables.ipynb**: dónde se ha realizado un estudio de los datos y un análisis de la variables respecto al output.
*  **2_ClasificacionEFS.ipynb**: dónde se han entrenado varios modelos de aprendizaje supervisado para realizar una clasificación binaria: predecir si el paciente va a volver a recaer después de un transplante de células madre.
*  **3_ClasificacionEFS_Time.ipynb**: dónde se ha implementado un modelo de regresión de Cox para estimar el tiempo que podría transcurrir hasta la aparición de dicho evento.
